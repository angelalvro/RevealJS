Funcionalidad

Se ha contruido un blog estatico con HTML y CSS. Dicho blog ofrece informacion sobre las tareas realizadas en la asignatura Difusion de contenidos multimedia via web.

Layout

EL layout general esta configurado de tal forma que disponemos principalmente de tres elementos.

Header

Ocupa todo el ancho de la página con un color de texto negro sobre un amarillo verdoso


.header {
	width: 100%;
	background-color: var(--amarilloVerdoso);
	text-align: center;
}

@media (min-width: 768px) {
	.header {
		display: flex;
		align-items: center;
		justify-content: center;
	}
}
